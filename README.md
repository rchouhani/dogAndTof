# 🐶 Dog & Tof — Site vitrine pour éducateur canin

Site vitrine responsive conçu pour offrir une présence en ligne professionnelle à un éducateur canin, avec une attention particulière portée à l’accessibilité et à la structure HTML sémantique.

---

## 🔎 Aperçu du projet

**Dog & Tof** est un site web statique permettant à un éducateur canin de présenter ses services, améliorer sa visibilité locale et faciliter la prise de contact avec ses futurs clients.

👉 Site en production :  
https://www.dogandtof.fr

---

## 🎯 Contexte et objectifs

Ce projet a été réalisé comme **side-project** en parallèle de mon apprentissage du développement fullstack.

### Objectifs personnels
- Consolider les fondamentaux **HTML & CSS**
- Travailler une structure web accessible et sémantique
- Déployer un site sur un hébergement traditionnel (OVH)

Le choix a volontairement été fait de rester sur une approche **frontend pure**, sans fonctionnalités dynamiques.

---

## 🛠️ Stack technique

- **HTML5**
- **CSS3**
- Site statique (sans framework)
- Hébergement : **OVH**

Aucune dépendance externe.

---

## ⚙️ Installation et lancement

Aucune installation nécessaire.

Le projet étant un site statique :

```bash
git clone git@github.com:rchouhani/dogAndTof.git
cd dog-and-tof
```
Puis ouvrir index.html dans un navigateur

---

## 🧠 Challenges & apprentissages
## Responsive design

Le principal défi a concerné le CSS afin de garantir une expérience cohérente sur différents écrans.

## Déploiement OVH

Le déploiement sur un hébergement OVH a nécessité une adaptation par rapport aux workflows modernes utilisés habituellement (ex : Vercel), notamment :
- upload manuel des fichiers
- gestion serveur plus traditionnelle

## Accessibilité

Travail volontaire sur :
- la structure HTML
- les attributs descriptifs des images et liens
